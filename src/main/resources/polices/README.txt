# Polices Unicode

Placez les polices Unicode suivantes dans ce dossier:

1. NotoSans-Regular.ttf
2. NotoSansMath-Regular.ttf  
3. NotoSansSymbols-Regular.ttf
4. NotoSansSymbols2-Regular.ttf
5. STIXTwoMath-Regular.ttf

Consultez le fichier FONTS_INSTALLATION.md à la racine du projet pour les instructions détaillées de téléchargement.

Sans ces polices, certains caractères Unicode pourraient s'afficher comme des carrés (□).
L'application fonctionnera quand même en utilisant les polices système disponibles.
